To run the program successfully, simply make sure all the files in the 
zip folder is extracted to the same folder. Afterwards open main.py and 
run the script. The program should then go through the video and the extracted 
images can be found in the folder "savedimages", with the corresponding YOLOv5txt 
files in the yolotxt folder. A sample video named "GL010031.MP4" has been provided
in the root folder.

The YOLOv5 network used for the evaluation, was cloned from the following
github link: https://github.com/ultralytics/yolov5
